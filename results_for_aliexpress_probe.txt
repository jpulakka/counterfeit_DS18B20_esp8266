--- # counterfeit_DS18B20_esp8266.ino
This is the November 2024 version of discover-classify_fake_DS18B20.
  We are in a game of whack-a-mole. At least six new clones have
  appeared on the market between 2019 and 2024. Please consider
  reporting suspicious test results to help keep the sketch current.
This sketch will test DS18B20 sensors attached to
  pin 4 for differences with the DS18B20 produced by
  Analog Devices / Maxim Integrated / Dallas Semiconductor.
  Details: https://github.com/cpetrich/counterfeit_DS18B20


Test DS18B20 sensors attached to GPIO 4.
Select the test:
  0. Enumerate attached sensors.

  1. Check temperature alarm function with a safe test according to datasheet.

  2. Discover clones with a safe test, using only function codes documented in the datasheet.
     This may be sufficient to detect clones.

  3. Classify clones with an agressive test, using undocumented function codes.
     This could conceivably mess up calibration parameters. Only use this on sensors
     dedicated to testing.
